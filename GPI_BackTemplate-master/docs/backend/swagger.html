get /reportes/actividad
Obtener reporte de actividad (reportesActividadGet)
Genera un reporte de actividad con los productos más buscados y las acciones realizadas

Return type

ReporteActividad

Example data

Content-Type: application/json
{
  "mensaje" : "Reporte de actividad generado correctamente",
  "actividad" : {
    "topProductosBuscados" : [ {
      "producto" : "Smartphone",
      "cantidad" : 480
    }, {
      "producto" : "Laptop",
      "cantidad" : 350
    }, {
      "producto" : "Tablet",
      "cantidad" : 220
    }, {
      "producto" : "Auriculares",
      "cantidad" : 180
    }, {
      "producto" : "Smartwatch",
      "cantidad" : 150
    } ],
    "acciones" : [ {
      "tipo" : "búsqueda",
      "cantidad" : 4800
    }, {
      "tipo" : "compra",
      "cantidad" : 1500
    }, {
      "tipo" : "clic en producto",
      "cantidad" : 3200
    } ]
  }
}

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.

Reporte de actividad obtenido correctamente ReporteActividad

get /reportes/ventas
Obtener reporte de ventas (reportesVentasGet)
Genera un reporte de ventas detallado

Query parameters

mes_inicio (optional)
Query Parameter — Mes de inicio del reporte (YYYY-MM)
mes_fin (optional)
Query Parameter — Mes de fin del reporte (YYYY-MM)
tipo_producto_id (optional)
Query Parameter — Filtrar ventas por tipo de producto (opcional)

Return type

ReporteVentas

Example data

Content-Type: application/json
{
  "mensaje" : "Reporte de ventas mensual obtenido",
  "reporte" : [ {
    "mes" : "2024-01",
    "monto_total" : 12500.5,
    "numero_transacciones" : 45,
    "monto_promedio" : 277.78
  }, {
    "mes" : "2024-02",
    "monto_total" : 9800,
    "numero_transacciones" : 38,
    "monto_promedio" : 257.89
  }, {
    "mes" : "2024-03",
    "monto_total" : 15000.75,
    "numero_transacciones" : 50,
    "monto_promedio" : 300.01
  } ]
}
