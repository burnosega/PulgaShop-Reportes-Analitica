Up 
get /reportes/productos/pdf
Generar reporte PDF de datos de ventas de productos (generarReportePDF)
Genera un archivo PDF con/sin filtros de todos los productos de la base de datos.

Query parameters

nombre (optional)
Query Parameter — Filtrar por nombre del producto (búsqueda parcial case-insensitive)
categoria (optional)
Query Parameter — Filtrar por categoría (puede contener múltiples categorías separadas por coma)
minPrecio (optional)
Query Parameter — Precio mínimo del producto format: float
maxPrecio (optional)
Query Parameter — Precio máximo del producto format: float
fechaInicio (optional)
Query Parameter — Fecha de inicio del rango de creación (formato ISO 8601) format: date-time
fechaFin (optional)
Query Parameter — Fecha de fin del rango de creación (formato ISO 8601) format: date-time
ordenarPor (optional)
Query Parameter — Campo para ordenar los resultados
orden (optional)
Query Parameter — Dirección del ordenamiento default: asc

Return type

byte[]

Example data

Content-Type: application/json
""

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/pdf
  • application/json

Reporte PDF generado exitosamente byte[]
